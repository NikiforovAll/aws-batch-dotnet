Amazon S3 (Simple Storage Service) is a scalable, high-speed, web-based cloud storage service designed for online backup and archiving of data and applications on Amazon Web Services (AWS). It provides object storage with a simple web service interface to store and retrieve any amount of data from anywhere on the web.

S3 is built to store and retrieve any amount of data at any time, from anywhere on the web. It's a simple storage service that offers an extremely durable, highly available, and infinitely scalable data storage infrastructure at very low costs.
